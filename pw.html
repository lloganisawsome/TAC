🌌 How to Set Up Ultraviolet

Ultraviolet is one of the most popular proxy frameworks. Here's how to deploy it for free:

1
Go to the List
Visit the Ultraviolet GitHub repo.
2
Pick a Proxy
Look for repos with the 🟡 JavaScript symbol — this means it's JavaScript-based and compatible.
3
Click the Green "Code" Button
On the GitHub repo page, click the big green button that says "Code".
4
Create New Codespace
Select "Create codespace on main" — it must say "main" NOT "X0", "Master", or anything else.
5
Install Dependencies
Type in the terminal: npm install
6
Start the Proxy
Once installation completes, type: npm start

📝 Doc Making
How to create your own proxy doc from scratch
1
Start with Google Docs
Google Docs is the standard — it's accessible on school devices and most admins won't immediately flag it as a gaming site.
2
Organize with headings
Use H1/H2 headings for sections. This makes the document scannable and lets viewers jump to the right section.
3
Use disguised link names
Instead of linking "Free Games Here" — name links "Math Practice", "Science Notes", etc.
4
Share carefully
Set sharing to "Anyone with the link can view". Don't send it directly to people who might report it.
5
Keep backups
Docs get taken down. Always have a backup copy ready.

🔍 Link Finding
How to find new proxy and game links
⭐
Method 1: GitHub Hunting
Browse GitHub for JavaScript repositories. Look for the 🟡 JS symbol and "Create codespace on main" option.
🌐
Method 2: Fastly CDN Links
Many proxy sites use "global.ssl.fastly.net" subdomains. Try searching: site:global.ssl.fastly.net games
📄
Method 3: Raid Other Docs
Check the "Other Docs" section — there are 100+ docs listed there. Each one has their own unique link collections.
🦆
Method 4: DuckDNS / Surge.sh / Bitbucket
Domains like duckdns.org, surge.sh, and bitbucket.io are often overlooked by school filters.
